# Движение денежных средств

Результат выполнения тестового задания

## Как запустить

```bash
python -m venv .venv
. .venv/bin/activate
pip install django
python src/manage.py runserver
```

### Технологии 
- Python, Django
- SQLite
- Шаблоны, HTML, CSS (Bootstrap)
